<h1>Overview</h1>

TemanNelayan (Manela) is a GEMASTIK XVIII project developed collaboratively with <a href="https://www.linkedin.com/in/alwan-athallah-mumtaz/" target="_blank">Alwan Athallah Mumtaz</a>, achieving **4th place** in the competition.

The platform helps fishermen identify potential fishing zones and probable fish species using satellite-derived oceanographic data, analytical processing, and AI-assisted recommendations.

<br />


## Responsibilities

- Conducted collaborative research on methodologies for predicting fish locations and species based on oceanographic indicators.
- Collected, processed, and analyzed::
<ol class="list-[lower-alpha] pl-20">
<li>Sea Surface Temperature (SST)</li>
<li>Chlorophyll-a (CHL-A)</li>
<li>Bathymetry data</li>
<li>Fish Zone Prediction</li>
</ol>

<Image src={SatteliteData} alt="Satellite Data"/>

- Utilized Google Earth Engine for large-scale satellite data acquisition.
- Processed and structured datasets using Python for analytical workflows.
- Built backend services using Node.js and Express.js.
- Designed and implemented offline-first system strategies.
- Implemented partial rendering techniques to improve usability on low-end devices.
- Participated in field testing and validation directly with fishermen.

## Impact

The project demonstrated how satellite data and lightweight AI-assisted systems could support practical fisheries decision-making in real-world maritime conditions.

Key outcomes included:
- Supporting fishermen with accessible fishing zone insights
- Enabling usability in low-connectivity environments
- Optimizing performance for lower-end mobile devices
- Achieved approximately 80% validation accuracy during field testing for core predictions and functionalities
- Securing 4th place at GEMASTIK XVIII 2025
